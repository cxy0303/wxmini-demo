# wxmini-demo
微信小程序简单开发框架demo

简单说明：  
1. 通过配置npm引入vant-ui库;  
2. 实现登录控制：  
    i. pages下每个页面引用components中base组件，此组件用于控制渲染界面和主题展示界面的显示;  
   ii. 一般加载一个页面如果此页面必须登录，则此页面登录前应该显示渲染界面，主题展示界面不应该加载，而不是简单的隐藏（因为隐藏组件还是会渲染，进入生命周期轮子）  
3. components  
     |--base ---------------公共组件base，每个页面引入此组件  
     |--pages---------------页面对应得主题组件  
     所以根目录下的pages组件一般来说不会有调用数据接口的代码
